# Documentation

This directory contains the working documentation for the Outlook PST MCP Server.

- `product.md`: product goal, constraints, non-goals, and user workflow.
- `architecture.md`: Go packages, service boundaries, data model, and runtime flow.
- `file-workflow.md`: how PST, `readpst`, EML files, SQLite, and exports are handled.
- `mcp.md`: MCP stdio behavior, tools, arguments, responses, and error handling.
- `development.md`: development workflow, Makefile targets, testing, build, and install behavior.
- `implementation-plan.md`: implementation plan used to build the current version.

All project-facing documentation is written in English.
